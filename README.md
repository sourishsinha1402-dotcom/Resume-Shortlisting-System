# AI-Powered Resume–Job Matching System

## Overview
This project builds an intelligent resume screening system using NLP and transformer-based embeddings to match candidates with job descriptions based on semantic similarity.

It goes beyond keyword matching by understanding contextual relevance, enabling more accurate and scalable hiring decisions.

## Objective
- Automate resume screening  
- Rank candidates based on job relevance  
- Improve recruitment efficiency using AI  

## Tech Stack
- Python  
- Sentence Transformers  
- Pandas, Scikit-learn  
- pdfplumber, docx2txt  

## Approach
1. Extract text from resumes (PDF/DOCX/TXT)  
2. Generate embeddings using transformer model  
3. Compute cosine similarity  
4. Rank candidates for each job description  

## Key Features
- Semantic (context-based) matching  
- Multi-format resume support  
- Automated ranking system  

## How to Run
- Open notebook in Google Colab  
- Upload resumes + job descriptions  
- Run all cells  

## Future Improvements
- UI dashboard  
- ATS integration  
- Model fine-tuning  

## Conclusion
Demonstrates practical use of NLP in recruitment analytics, enabling intelligent and scalable candidate screening.
